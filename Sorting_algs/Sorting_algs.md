## Selection Sort
In order to sort a list, selection sort requires traversing through that list of items multiple times to pull out specific elements one at a time. Because of this, this algorithm is neat, but it is not fast.

Worst Case: O(n²)
Best Case: O(n²)

[See Example](selection_sort.py)

## Insertion Sort
Insertion sort works to repeatedly move each element in an unsorted list into its proper position by comparing the current element to elements before it. As the previous elements are compared to the current element, they are moved to the right until the correct position is found for the current element. Once the position is found the element is inserted at that position.

Worst Case: O(n²)
Best Case: O(n)

[See Example](insertion_sort.py)
